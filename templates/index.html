About Project

Purpose of the system

Collect Clothes

Donors can submit clothing details, quantity, and pickup address using a web form.

Manage Requests

Needy families or NGOs can request clothes according to category, urgency, and number of people.

Track Records

All submissions are stored in a SQLite database and displayed in the admin panel.

Workflow

How this Python website works

01

Donation Entry

The donor fills in name, mobile number, clothing type, quantity, and address.

02

Data Storage

Flask receives the form data and stores it in the SQLite database.

03

Request Submission

Applicants submit their needs using the request form on the same website.

04

Admin Review

The admin page displays donation and request records for management.
